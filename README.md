# Store_sales_Forecasting_Practice
This project looks at historical sales data from a global superstore to understand how the business performs across products, regions, and customer segments. It also includes a basic 7-day sales forecast based on recent trends.

The aim was to take raw data, clean and explore it, and turn it into something meaningful through insights and an interactive dashboard built in Power BI.

## What I was trying to solve
 Retail businesses depend on sales data to make decisions around products, regions, and planning. This project focuses on answering questions such as:

Which products actually drive the most sales?
Which regions are performing better than others?
How do different customer segments behave?
What do sales trends look like over time?
Can short-term sales be reasonably forecasted from historical patterns?

## Data Preparation
 The dataset was first reviewed and cleaned in Excel to make sure it was usable for analysis.
 
 This involved:
Checking the structure and consistency of the data
Removing duplicate entries
Handling missing or incomplete values
Making sure data types were correctly formatted

## Exploratory Data Analysis (EDA)
Once the data was clean, I explored it to understand what was actually driving sales.
Some of the key areas of focus included:

Sales performance by city and region
Best and worst performing product categories
Yearly and quarterly sales trends
Customer segment behaviour
Identifying patterns in sales over time
Building a simple 7-day sales forecast based on historical trends

This step helped highlight where the business is performing well and where there are clear differences in demand.

## Dashboard (Using Power BI)
 This project helped turn a raw retail dataset into clear insights around performance and trends, and showed how historical data can be used to make short-term forecasts. It demonstrates a complete end-to-end analytics workflow from cleaning and exploration through to dashboard delivery.

 ## Tools Used
Excel – data cleaning and preparation

Power BI – data visualisation and dashboard building
